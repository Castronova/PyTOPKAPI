Following the latest source¶

These are the instructions if you just want to follow the latest PyTOPKAPI source, but you don’t need to do any development for now.

The steps are:

  • Install git
  • get local copy of the git repository from github
  • update local copy from time to time

Get the local copy of the code¶

From the command line:

git clone git://github.com/sahg/PyTOPKAPI.git

You now have a copy of the code tree in the new PyTOPKAPI directory.

Updating the code¶

From time to time you may want to pull down the latest code. Do this with:

cd PyTOPKAPI
git pull

The tree in PyTOPKAPI will now have the latest changes from the initial repository.
